Compilation:
============
1. In NurSuhaira_5841549_A1/Part6:
	javac streamCipher.java 
	// to compile streamCipher.java to generate streamCipher.class

Before Execution:
=================
1. Make sure streamCipher.java is compiled with no error.
2. In command prompt:
	[streamCipher.java]
		java streamCipher <type> <text> <key>
	   e.g. java streamCipher encrypt WOLLONGONG 3
	   e.g. java streamCipher decrypt MQJJ 3
	
During Execution:
=================
1. A display message to prompt user that keystream values are being generated using user's key input
2. A display of type, text and key used
3. If encrypted, a ciphertext will be generated along with the plaintext origin
4. If decrypted, a plaintext will be generated along with the ciphertext origin

Execute Environment
===================
Operating system: Ubuntu
Using terminal.
